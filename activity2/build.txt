AWTUI
--------------------------------------------------------------------
AWTUI provides a graphical user interface using AWT.
--------------------------------------------------------------------
How to Compile
To compile AWTUI you must use javac in the terminal.
1. Type "javac AWTUI.java" without quotations to compile the java file.
--------------------------------------------------------------------
How to Run
1. Change directories using Git Bash (cd) to activity2 folder.
2. Type "java AWTUI.java" without quotations to run the program
